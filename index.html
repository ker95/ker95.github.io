如何建立一个博客

1.下载node.js

从 node.js官网 下载适合自己电脑的版本，一般会给你推荐两个版本
这两个版本分别是稳定版和测试版，一般来说我们都是选择第一个稳定版，
下载完之后安装在自己的电脑上就行了，如果你不知道是否安装了node.js
可以再终端输入

如果你的电脑上安装了node.js系统会给你自动弹出当前的版本号

2.安装hexo

hexo是基于node.js的博客框架，以后我们博客的大部分操作都是要通过hexo来执行的。下载hexo 通过以下代码来执行系统自己带代码我就省略了，只写出手写代码，代码如下：

npm install -g hexo
sudo npm install -g hexo

这里写了两种方法，第一种是正常安装，第二种就是权限安装，当你用第二种方法时就要输入你的用户密码，再你输入密码的时候，系统是不会显示的，密码输入完成就直接按下 “enter” 键，就OK了。
更新hexo的执行命令是

npm update hexo -g

这样就更新到hexo最新版了

3.创建博客

首先把终端控制台调整到桌面，创建一个名字为“博客”的文件夹

mkdir 博客

然后输入进入这个新建的文件夹，进行初始化设置

hexo init

这样系统就会给你设置了几个默认文件夹

下面来说说怎么这些文件夹的作用

首先是 “_config.yml”，打开这个文件，先看这张图片这些是系统默认的，如果你不喜欢可以自己修改

titile:标题 subtitle：副标题 description：描述 author：作者 ，这些都是直译过来的也没什么特殊的

language：默认为英文，如果想要变成中文，只要加上 zh-CN 就行了

到最后面有这么几行代码

theme：这个是放博客主题的，系统自带的主题就是landscape，你也可以自己下载主题，把下载的主题放在 “themes” 文件夹下，然后把landscape改为你下载的文件夹名称就可以了

deploy 和 type 这个就是用来关联github的，怎么关联等我们更到github得时候再给大家详细说一下

再来说说 “source”文件夹，在这个文件夹下找到 “_posts”文件夹，这个文件夹就是放你的博客的地方，像这里就由一篇名字为“hello-world”的默认博客

接下来说说怎创建博客

hexo new “博客名称”
#### 引号里面的就是你的博客名称，自己可以随便命名

然后就可以打开这个博客

这是刚刚建立的一个空博客，上面的几行相信大家都明白，tags就是这篇博客的标签，在这里面你可以加入行内样式来时布局更好

从第六行开始就是你的博客内容那个了，这里面都是支持行内属性的，所以可以自己对布局进行设计

最后当你这篇博客写完了要保存，保存了之后要进行这么两条语句，非常重要，不进行这两条语句的话就没法同步到你的博客

hexo g
hexo d

执行完这两句代码后执行这条语句

把红线圈起来的部分放到地址栏打开就可以看到你的博客了，不过这只是本地文件，还没有上线，关于如何上线，这就要用到github了，我们后面就告诉大家怎么关联github~~

mac终端的一些小技巧

在终端的代码中每一条命令前面都要加上一个空格

如果一条命令有两部分组成，在两部分中间也要加上空格

下面就列举几个简单的命令代码：

  • 查看当前文件下的所有子文件：ls；
  • 找到当前文件下的子文件： cd 子文件名；（只能查找子集的文件，不能越级查找）
  • 返回上一级： cd ..
  • 返回上两级： cd ../..
  • 创建文件夹： mkdir 文件夹名称
  • 创建文件： touch 文件名称（记得加上后缀）
  • 删除文件： rm 文件名（这种方法并不能删除文件夹）
  • 删除文件夹： rm -rf 文件名称（既可以删除文件，也可以删除文件夹）
  • 查看当前文件路径： pwd
  • 查看IP地址： ifconfig
